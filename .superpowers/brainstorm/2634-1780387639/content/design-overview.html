整体设计方案概览

以下是对整个 App 的设计构思，逐块确认

🏗 技术架构

语言/UI: Kotlin + Jetpack Compose
本地数据库: Room（SQLite）
架构模式: MVVM（Model-View-ViewModel）
导航: Navigation Compose + Bottom Navigation（4 标签）
图表: Vico（Compose 原生图表库）

📱 导航结构

📈

📋

设置

💾 数据模型

Exercise (动作库) — id, 名称, 部位, 图标名, 是否预设
Workout (训练记录) — id, 日期, 开始时间, 备注
Set (训练组) — id, workoutId, exerciseId, 组号, 重量, 次数, 组间休息
WorkoutPlan (训练计划) — id, 名称, 星期X, 动作列表
BodyMetric (身体数据) — id, 日期, 体重, 体脂率, 备注

🖼 页面概览

🏋️ 训练页
开始训练 → 选动作 → 点按记录组数/次数/重量 → 结束训练
📈 进度页
力量增长曲线、训练频率、体重变化趋势
📋 计划页
查看/创建训练计划，可一键按计划开始训练
⚙️ 设置页
身体数据录入、动作库管理、单位设置、数据导出
